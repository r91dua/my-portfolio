Projects

3rd Party Integration & API Platform

Developed Ford’s enterprise-grade 3rd Party Platform to onboard external partners such as Amazon for the In-Car Delivery pilot. Built a secure API marketplace enabling vendors to integrate with Ford Connected Vehicle data and services. Technologies used: API Gateway, OAuth2, Azure, and AWS integrations.

FordPass & Lincoln Way Voice Assistants

Created Alexa Skills and Google Actions for FordPass and Lincoln Way, enabling customers to perform remote vehicle commands (start, lock/unlock, charge status). Integrated with Ford Connected Vehicle APIs and voice assistant ecosystems. Technologies used: Node.js, AWS Lambda, Alexa Developer Console, Google Actions SDK.

Enterprise Vehicle Enrollment Platform

Designed and implemented a global enrollment system that enables feature subscriptions for retail and fleet customers (Phone as a Key, Connected Navigation, Voice, and Ford BlueCruise). Delivered high availability and resilience using Azure and Pivotal Cloud Foundry with CI/CD pipelines.

Fleet Data Management & Insights Platform

Enabled connected vehicle data for fleet management systems, allowing customers to monitor fuel levels, maintenance status, and telematics in real-time. Supported data-driven decision-making for improved fleet operations. Technologies used: Azure Data Services, REST APIs, and telematics integration.

Otis DevSecOps Automation Framework

Established a standardized DevSecOps process for Otis global applications. Automated infrastructure provisioning, application delivery, and CI/CD pipelines with integrated security scanning and compliance checks. Technologies used: Terraform, GitHub Actions, SonarCloud, Coverity, Azure DevOps.
